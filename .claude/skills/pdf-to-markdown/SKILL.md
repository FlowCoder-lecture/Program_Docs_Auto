# PDF to Markdown

## Overview

PDF 형식의 공식 문서(사업계획서, 정부지원사업 신청서, 제안서, 보고서 등)를 분석하여 구조화된 마크다운 문서로 변환한다. 문서의 레이아웃, 표, 섹션 구조를 보존하면서 재사용 가능한 템플릿을 생성한다.

## Workflow

### Step 1: PDF 분석

Read 도구로 PDF 파일을 읽고 다음을 파악:

**문서 유형 식별**
- 사업계획서 (Business Plan)
- 정부지원사업 신청서 (Government Application)
- 제안서 (Proposal)
- 보고서 (Report)
- 계약서/협약서 (Contract)
- 기타 양식 (Form)

**구조 요소 파악**
- 제목/부제목 계층
- 섹션/챕터 구분
- 표 (Table)
- 입력 필드 (빈칸, 괄호)
- 체크박스/선택항목
- 목록 (번호/글머리)

### Step 2: 구조 매핑

PDF 요소를 마크다운으로 매핑:

| PDF 요소 | 마크다운 |
|----------|----------|
| 문서 제목 | `# H1` |
| 대제목 | `## H2` |
| 중제목 | `### H3` |
| 소제목 | `#### H4` |
| 표 | `\| 테이블 \|` |
| 입력란 `[ ]` | `**[입력: 설명]**` |
| 체크박스 `□` | `- [ ] 항목` |
| 번호 목록 | `1. 2. 3.` |
| 글머리 기호 | `- 항목` |
| 강조 | `**굵게**` |
| 구분선 | `---` |

### Step 3: 변환 규칙 적용

#### 메타데이터 (문서 상단)
```yaml
---
title: "문서 제목"
type: "사업계획서 | 신청서 | 제안서"
source: "원본파일명.pdf"
converted_date: "YYYY-MM-DD"
total_pages: N
---
```

#### 입력 필드 표현
```markdown
<!-- 단일 입력 -->
**[입력: 필드명]**

<!-- 여러 줄 입력 -->
**[입력 - 여러 줄: 설명]**

<!-- 선택 항목 -->
**[선택: 옵션1 / 옵션2 / 옵션3]**

<!-- 날짜 -->
**[날짜: YYYY-MM-DD]**

<!-- 금액 -->
**[금액: 원]**
```

#### 표 변환
```markdown
| 항목 | 내용 | 비고 |
|------|------|------|
| 기업명 | **[입력: 기업명]** | 필수 |
| 대표자 | **[입력: 대표자명]** | 필수 |
```

#### 체크리스트
```markdown
- [ ] 사업계획서 1부
- [ ] 법인등기부등본 1부
- [ ] 사업자등록증 사본 1부
```

#### 서명란
```markdown
| 구분 | 서명 | 날짜 |
|------|------|------|
| 신청인 | **[서명]** | **[날짜]** |
```

### Step 4: 출력 생성

#### 파일 명명
```
[원본파일명]_converted.md
```

#### 출력 구조
```markdown
---
title: "2025년 창업지원사업 신청서"
type: "신청서"
source: "application_2025.pdf"
converted_date: "2025-12-29"
---
```

## 특수 케이스 처리

### 병합 셀 표
```markdown
<!-- 원본: 병합된 셀 구조 -->
| 구분 | 상세 | 금액 |
|------|------|------|
| **인건비** | | |
| | 대표자 급여 | 1,000,000 |
| | 직원 급여 | 2,000,000 |
```

### 이미지/도표
```markdown
> 📊 **[도표: 시장 규모 그래프]**
> - 위치: 원본 3페이지
> - 설명: 연도별 시장 규모 변화
```
