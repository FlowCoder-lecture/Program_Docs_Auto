## 4. 사업 아이템

### 4.1 사업 개요

- **서비스명**: LearnAI (런에이아이)
- **한 줄 소개**: AI 기반 개인 맞춤형 학습 플랫폼
- **핵심 가치 제안**: 학생 개개인의 학습 수준과 패턴을 AI가 분석하여 최적의 학습 경로와 콘텐츠를 제공
- **타겟 고객**: 중·고등학생 및 학부모, 학원/교육기관
- **해결하는 문제**: 획일화된 교육으로 인한 학습 효율 저하, 개인별 취약점 파악 어려움

### 4.2 상세 설명

LearnAI는 AI 기술을 활용하여 학생 개개인의 학습 데이터를 실시간으로 분석하고,
각자의 수준과 학습 스타일에 맞는 최적의 학습 경로를 제시하는 에듀테크 플랫폼입니다.

기존 온라인 교육이 모든 학생에게 동일한 콘텐츠를 제공하는 것과 달리,
LearnAI는 학생의 문제 풀이 패턴, 오답 유형, 학습 시간대, 집중도 등을 종합 분석하여
개인화된 커리큘럼을 자동 생성합니다.

또한 AI 튜터가 24시간 질의응답을 지원하며, 학부모와 교사에게는
학생의 학습 현황과 성장 리포트를 제공합니다.

### 4.3 핵심 기능

**1. AI 학습 진단**
- 학생의 현재 수준을 정밀 진단
- 취약 단원 및 개념 자동 식별
- 차별점: 단순 정오답이 아닌 풀이 과정 분석

**2. 맞춤형 학습 경로**
- 개인별 최적화된 커리큘럼 자동 생성
- 실시간 난이도 조절
- 차별점: 강화학습 기반 적응형 알고리즘

**3. AI 튜터**
- 24시간 질의응답 지원
- 단계별 힌트 및 풀이 설명
- 차별점: GPT-4 기반 자연스러운 대화형 학습

**4. 학습 분석 대시보드**
- 학습 패턴 및 성과 시각화
- 학부모/교사용 리포트 자동 생성
- 차별점: 예측 분석을 통한 성적 향상 로드맵

### 4.4 기술 스택

- Frontend: React, Next.js, TypeScript
- Backend: Python, FastAPI, Node.js
- Database: PostgreSQL, MongoDB, Redis
- AI/ML: PyTorch, Hugging Face, LangChain
- Infra/Cloud: AWS (EC2, S3, Lambda), Docker, Kubernetes
- 기타: Elasticsearch, RabbitMQ
